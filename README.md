# golang-fave
```
Usage of ./fave:
  -dir string
    	virtual hosts directory
  -host string
    	server host (default "0.0.0.0")
  -port int
    	server port (default 8080)
```

## Development
`make` - debug, `make run` - run on all ips and port 8080, `make build` - build release for Linux, Mac OS X and Windows.
